# リリース前チェックリスト（主要機能の手動確認）

リファクタやデプロイ前に、以下をブラウザで手動確認する。

## ダッシュボード (/dashboard)

- [ ] 画面が表示され、カード（期限切れ・今日・今週 High）が表示される
- [ ] カレンダー表示切替（日 / 週 / 月）が動く
- [ ] カレンダーの前後ナビ・「今日」で日付が変わる
- [ ] 日付をクリックすると DayTasksDrawer が開く
- [ ] タスク行のチェックで完了/未完了が切り替わる
- [ ] 完了操作後に「元に戻す」Toast が表示され、クリックで取り消せる
- [ ] タスクタイトルクリックで TaskDrawer（編集）が開く
- [ ] 進捗セクション（グループ / プロジェクト切替、今週/今月）が表示される

## ワークスペース (/workspace)

- [ ] 画面が表示される
- [ ] 表示モード切替（グループボード / 単一グループ / プロジェクトボード）が動く
- [ ] フィルタ（期限・状態・優先度・Bucket 等）で一覧が絞り込まれる
- [ ] タスクの表示・編集・削除・完了トグルができる

## Planning (/planning)

- [ ] 画面が表示される
- [ ] 表示切替（日 / 週 / 月 / 年）が動く
- [ ] フィルタでタスクが絞り込まれる
- [ ] タスク操作ができる

## 設定 (/settings)

- [ ] 画面が表示される
- [ ] 「JSONエクスポート」でバックアップがダウンロードされる
- [ ] 正しいエクスポート JSON を選択して「インポート実行」でデータが復元される
- [ ] インポート時の警告が表示される

## マスタ（グループ・プロジェクト・Bucket）

- [ ] ダッシュボードまたは該当画面からマスタ編集が開き、一覧が表示される
- [ ] グループ / プロジェクト / Bucket の追加・編集・削除ができる
- [ ] プロジェクトのグループ紐づけができる

## 繰り返しテンプレート

- [ ] テンプレート一覧が表示される
- [ ] 作成・編集・有効/無効ができる
- [ ] テンプレートに紐づくタスクが期待どおり生成される（必要なら確認）

## CI

- [ ] `npm run lint` が通る
- [ ] `npm run typecheck` が通る
- [ ] `npm test` が通る
- [ ] `npm run build` が成功する
